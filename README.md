# .bashrc

### Get with

`wget https://raw.githubusercontent.com/AmauMarshall/bash-utils/main/.bashrc`

### or clone repo and extract:

```bash
cd ~
git clone https://github.com/AmauMarshall/bashrc.git
cp bashrc/.bashrc .
# clear clone folder:
rm -rf bashrc
```

### Update your current bash session

`. .bashrc`
